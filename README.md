## Deskripsi Program
Program ini dibuat untuk membantu pengolahan data nilai mahasiswa menggunakan
konsep fungsi dan modularisasi dalam bahasa Python.

Pengguna memasukkan sejumlah nilai mahasiswa, kemudian program akan memproses
data tersebut untuk mendapatkan hasil perhitungan yang dibutuhkan.

## Studi Kasus
Dalam studi kasus ini, program digunakan untuk:
1. Menerima input nilai mahasiswa
2. Menghitung rata-rata nilai
3. Menentukan status kelulusan mahasiswa
4. Menampilkan nilai tertinggi

Agar program lebih terstruktur dan mudah dipahami, fungsi-fungsi perhitungan
dipisahkan ke dalam file terpisah dan dipanggil oleh program utama.

## Struktur File
- `main.py` : Program utama untuk input dan output
- `utils.py` : Berisi fungsi-fungsi perhitungan

## Cara Menjalankan
1. Jalankan file `main.py`
2. Masukkan jumlah nilai mahasiswa
3. Masukkan nilai sesuai permintaan
4. Program akan menampilkan hasil perhitungan
